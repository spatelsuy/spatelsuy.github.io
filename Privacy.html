Privacy Policy

This privacy policy explains how data is handled in this app, blog, and associated learning tools provided through direct URLs or mobile platforms like the Google Play Store.

1. Data Collection

This platform does not require user authentication and does not actively collect personal information such as names, emails, or addresses.

2. Use of Input Data

Some learning tools (e.g., text summarization, PDF merging, speech-to-text conversion) may ask users to provide text or files. These inputs are used temporarily to perform the requested action. No data is stored, logged, or shared.

3. Third-party APIs and Libraries

Some features use open-source models and APIs (e.g., Hugging Face Transformers, Python GenAI libraries). Data may be passed to these libraries for processing. These libraries typically do not retain data, but users should avoid uploading sensitive or confidential information.

4. Analytics and Tracking

This platform does not use analytics, cookies, or tracking technologies.

5. External Links

There are links to other websites, GitHub repositories, and tools. These external sites operate independently and have their own privacy policies.

6. Data Security

Reasonable steps are taken to ensure the safety of your input during processing. However, for tasks involving sensitive data, it is advised not to use public tools.

7. Children's Privacy

This app is not intended for children under 13. No content is targeted specifically at children.

8. Changes to This Policy

This privacy policy may be updated from time to time. Any changes will be posted on this page with the updated date.

9. Contact

For questions or concerns about this policy, reach out via email or through social platforms listed in the app.
