audio-to-text conversion system

An audio-to-text conversion system using Whisper and FFmpeg on Windows. This program leverages the power of the Whisper model from OpenAI for accurate transcription, combined with FFmpeg for audio processing, all within a Python environment. This solution works completely offline, meaning no sensitive audio files or transcriptions are uploaded to the cloud — keeping content private and secure, ensuring data privacy and security.
